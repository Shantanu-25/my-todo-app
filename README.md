# 📝 Todo App

A simple and intuitive Todo App built with Python and Streamlit to help users manage daily tasks efficiently.

## Features

* ➕ Add new tasks
* ✏️ Edit existing tasks
* ✅ Mark tasks as completed
* 📋 View all pending tasks
* 💾 Persistent task storage using text files
* 🌐 Simple web interface powered by Streamlit

## Tech Stack

* Python
* Streamlit
* File Handling

## Installation

1. Clone the repository:

```bash
git clone https://github.com/Shantanu-25/my-todo-app.git
cd my-todo-app
```

2. Create and activate a virtual environment:

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run web.py
```
